# Student Management System

A simple Student Management System built with Python.

## Features

- Add Student
- Delete Student
- Update Student
- Search Student
- Show All Students
- Save Data to JSON

## Technologies

- Python
- JSON
- Object-Oriented Programming (OOP)

## Project Structure

```
student-system/

├── main.py
├── student.py
├── manager.py
├── students.json
└── README.md
```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/alisaraei/Student-Management-System.git
```

2. Go to project folder

```bash
cd Student-Management-System
```

3. Run

```bash
python main.py
```
